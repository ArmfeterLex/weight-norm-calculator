# Weight Norm Calculator

Модуль для консольных приложений на C#, рассчитывающий норму веса по формуле "рост минус 100" с допустимым отклонением ±10%.

## Функциональность

- Ввод роста (см)
- Ввод веса (кг)
- Расчет идеального веса: рост - 100
- Допустимый диапазон: ±10% от идеального
- Вывод рекомендации: норма / похудеть / поправиться

## Пример работы

Введите ваш рост (в см):
180
Введите ваш вес (в кг):
80
Ваш вес: 80 кг. Норма

Введите ваш рост (в см):
170
Введите ваш вес (в кг):
85
Ваш вес: 85 кг. Нужно похудеть

Введите ваш рост (в см):
175
Введите ваш вес (в кг):
60
Ваш вес: 60 кг. Нужно поправиться

## Куда можно встроить

- .NET Core / .NET Framework
- Консольное приложение

## Примечание

Используется упрощенная формула без учета возраста, пола и телосложения.
